Preferences

Preference data may is saved to the device so that the user can customize sessions and provide a personalized experience. The following parameters are customizable.

Session Length

Default: 20 minutes

Length of meditation sessions.

Enable Foreground Image

Default: False

If enabled, a foreground image is displayed on the meditation screen. Its intensity is controlled by a selectable parameter (by default itâ€™s skin temperature).

Audio Track

Default: None

Selects between several pre-recorded audio tracks to play during the meditation session.

Meditation Image Theme

Default: Sunset

Selects the image set which is displayed to the user during meditation sessions.

Show Connection Messages

Default: True

If enabled, BioZen will show toast messages each time any sensor connects or disconnects.

Background Image Control

Default: Delta

This option brings up a list of biosensor parameters which allows you to select which parameter controls the background image.

Foreground Image Control

Default: Skin Temperature

This option brings up a list of biosensor parameters which allows you to select which parameter controls the foreground image.

User Mode

Default: Single User Mode

This option allows you to select between Single User Mode and Provider Mode.

Show Help on Start

Default: True

If enabled the main menu page will also show scrollable pages of help screens.

Show Help on View

Default: True

If enabled the "live view" page will be preceded by a help page.

Show Help on Review

Default: True

If enabled the "Review" page will be preceded by a help page.

Show Help on New Session

Default: True

If enabled the "New Session" page will be preceded by a help page.

xxx Device Names

Default:

Use these fields to enter non-default Bluetooth names for devices. Default names are as follows: NeuroSky products: "MindSet", Zephyr Bioharness: "BH *", Shimmer: "RN*"

Advanced Preferences

The advanced preferences should not be changed by the user in normal circumstances.
